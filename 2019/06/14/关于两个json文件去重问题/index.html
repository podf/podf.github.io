关于两个json文件去重问题

Posted on 2019-06-14 | In javascript |

解决思路

通过filter判断a中是否存在b的数据

知识点

filter函数

filter() 方法创建一个新的数组，新数组中的元素是通过检查指定数组中符合条件的所有元素。

find函数

find()可以检索数组中满足指定条件的第一个元素。
如果检索到满足条件的元素，那么返回此元素，否则返回undefined。
